# pet_care_ls

一个基于 **Next.js (App Router)** + **React** + **TypeScript** + **Tailwind CSS** 的宠物护理/门店落地页项目。

## 预览与功能

- 落地页组件：`components/landing/*`
- 静态资源：`public/assets/*`
- Next.js 路由入口：`app/*`

## 快速开始（本地运行）

### 环境要求

- Node.js：建议 18+（或 20+）
- 包管理器：npm

### 安装依赖

```bash
npm install
```

### 启动开发环境

```bash
npm run dev
```

然后打开：`http://localhost:3000`

### 构建与生产运行

```bash
npm run build
npm run start
```

## 常用命令

```bash
npm run lint
```

## 环境变量（如需）

本项目已在 `.gitignore` 中忽略所有 `.env*` 文件。若你后续需要配置环境变量，请在项目根目录新建例如：

- `.env.local`

并将敏感信息保存在本地，不要提交到 GitHub。
